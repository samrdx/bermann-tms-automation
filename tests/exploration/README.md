# Exploration Scripts

Scripts para inspeccionar páginas y extraer selectores.

## Uso
```bash
npm run test:explore:contratos
npm run test:explore:viajes
npm run test:explore:asignar
```

## Propósito

- Extraer selectores (inputs, dropdowns, buttons)
- Tomar screenshots
- Documentar estructura de página
- Agregar selectores a Confluence

## Output

- JSON con selectores en consola
- Screenshots en `./reports/screenshots/`
- Logs en `./logs/app.log`
